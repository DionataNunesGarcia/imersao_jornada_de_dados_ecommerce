# Coding Rules

## Python

- usar type hints
- funções pequenas
- evitar duplicação

## Streamlit

- usar st.set_page_config(layout="wide")
- usar st.sidebar para navegação
- usar st.columns para KPIs

## SQL

apenas SELECT.

nunca alterar dados do banco.

## Código

sempre gerar:

- imports
- funções auxiliares
- tratamento de erro
